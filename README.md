a. var and let create variables that can be reassigned another value.
b. const creates "constant" variables that cannot be reassigned another value.
c. developers shouldn't use var anymore. They should use let or const instead.
d. if you're not going to change the value of a variable, it is good practice to use const.

map(), filter(), and forEach() are three functions in JavaScript that are used for array manipulation. However, they are all used for different purposes.

forEach(): This function simply goes through all the items in the array and performs the desired action on them. No new array is produced; the main use of this function is when you simply need to perform a certain action for all the items in the array.

map(): This function is similar to the forEach() function in the sense that it goes through all the items in the array and performs the desired action. However, the main difference is that the map() function will produce a (new array) with the desired action performed on each of the items in the array.
